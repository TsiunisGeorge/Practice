# Задания
1. Минимальное API
     Создайте конечную точку с помощью контроллера, которая возвращает текущее время сервера. Изучите отличие от подхода с MapGet().
2. Чтение заголовков
     Создайте middleware, который проверяет наличие кастомного заголовка X-API-Key. Если его нет, возвращает статус 403 Forbidden.

В проекте выполнены оба задания поэтому можно его проверить двумя способами
curl -H "X-API-Key: 123" http://localhost:5123/api/time
Ответом должно являться время в формате UTC

curl http://localhost:5123/api/time
Ответ: Forbidden: Missing X-API-Key header

Различия контроллеров и Map.Get()
Архитектурно контроллеры часть архитектуры MVC и подрузумевает разделение приложения на слои, более маштабируемый подход, minimal API же это самый простой способ написать API, плохо маштабируется.
Так же minimalApi не реализует некоторые функции имеющиеся в контроллерах например фильтрация, автоматическая валидация и тп.
